Грид-области

Свойство CSS grid является сокращённой формой записи, которая устанавливает значения для всех явных свойств сетки (grid),всех неявных свойств сетки (grid), и свойств для промежутков между рядами и столбцами и сетки в одной строчке.

CSS Grid это новая модель для создания шаблонов, оптимизированная для создания двумерных макетов. Она идеально подходит для: шаблонов сайтов, форм, галерей и всего, что требует точного и отзывчивого позиционирования.

Грид-ячейка – наименьшая часть на гриде. Концептуально она похожа на ячейку таблицы. Грид определён для родительского элемента, дочерние элементы автоматически размещаются в каждой ячейке заданного грида.

Элементы могут занимать одну или несколько ячеек внутри строки или колонки, таким образом, создаётся грид-область. Грид-области должны быть перпендикулярными.
